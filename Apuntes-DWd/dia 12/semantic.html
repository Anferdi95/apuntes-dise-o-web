Elementos con significado!

Ejemplos de etiquetas semanticas: Table, Form, Article

HEADER

Es el encabezado de una página, sección o artículo. Incluye elementos como el logo, texto introductorio, menus de navegacion. Puede haber multiples headers en una pagina, pero no opuede haber un header dentro de un footer, o de otro header.

FOOTER

Es el pie de página de un sitio o sección. Puede haber múltiples por página. Puede incluir información como:
  • Información de contacto
  • Enlaces de RRSS
  • Mapa del Sitio
  • Creditos
  • Formulario de Contacto

NAVEGACIÓN

Define un conjunto de enlaces de navegación. No todos los links deben estar dentro de un "Nav".

MAIN

Crea un contenedor para el contenido principal de nuestra página. Este debe ser único y no debe incluir contendio repetido de otras secciones, como por ejemplo del sidebar, logos, etc. No puede ser hijo de etiquetas como: Article. aside, footer, header o nav.

ASIDE

Define el contenido relacionado indirectamente con el contendio principal, como por ejemplo un sidebar. Elementos que solemos poner en un aside:
  • Publicidad
  • Biografía de autor
  • Enlaces relacionados
  • Información de Perfil

SECTION

Define una sección en una página donde se agrupa los contenidos relacionados, y normalmente incluyen un título.
Una web se compone de varias secciones, cada una con su contenido y propósito.
  • Cápitulos
  • Introducción
  • Noticias
  • Información de Contacto

ARTICLE

Define un contenido INDEPENDIENTE. El mismo puede ser compartido o reutilizarlo en otro sitio sin requerir contexto.
  • Post de Blóg
  • Artículo de Revista
  • Comentario de un Usuario
  • Tarjeta de Producto

Puedo encontrar artículos dentro de secciones o secciones dentro de artículos.
Un artículo puede tener múltiples secciones. Una seccion puede tener Múltiples artículos.

FIGURE

Define elementos autocontenidos, como puede ser una imágen, un diagrama, un gráfico, una tabla o una foto.
Dentro de ella, se suele utilizar la etiqueta "figcaption" para describir el contenido, esto es ademas del ya incluido "alt" en caso de ímagenes.

MARK

Define un texto resaltado, normalmentese muestra con un Fondo Amarillo
